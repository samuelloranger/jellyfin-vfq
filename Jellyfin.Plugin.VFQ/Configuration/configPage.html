VFQ Auto Selector

Automatically selects French Canadian (VFQ / FR-CA) audio tracks when a movie starts playing.

When enabled, the plugin will automatically switch to a VFQ audio track if one is detected when playback starts.
When multiple VFQ tracks exist, prefer the one with the best codec (TrueHD > DTS-HD MA > FLAC > EAC3 > DTS > AC3 > AAC) and most channels.

Detected Keywords

The plugin identifies VFQ tracks by looking for these keywords in audio track titles and language tags:

  • VFQ - Version Française du Québec
  • FR-CA / FRA-CA / FRE-CA - French Canadian language codes
  • Québécois / Quebecois
  • French Canadian / Français Canadien / Français (Canada)
  • QC - Quebec abbreviation
